Bienvenue sur le Sélecteur Aléatoire

Cette application utilise le stockage local de votre navigateur, afin d'enregistrer les éléments que vous entrez.

C'est pour cela que nous vous demandons d'utiliser ce site dans votre navigateur habituel et en dehors du mode privé.


De plus aucune donnée personnelle n'est collectée, tout est stocké sur votre appareil.

Merci de votre compréhension.
